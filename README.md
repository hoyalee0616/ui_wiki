# Utility Wiki Codex Kit

이 폴더는 코덱스로 바로 UI 구현을 시작할 수 있도록 만든 기준 자료 모음이에요.

## 포함 파일
- `utility-wiki-mobile-reference.png` : 모바일 UI 참고 시안
- `utility-wiki-desktop-reference.png` : PC UI 참고 시안
- `STYLE_GUIDE.md` : 스타일 가이드 / 컴포넌트 규칙 / 토큰 / 코덱스용 구현 지침

## 사용 방법
1. 이미지 2장을 먼저 보면서 전체 무드를 맞춥니다.
2. `STYLE_GUIDE.md`의 디자인 토큰과 레이아웃 규칙을 기준으로 컴포넌트를 나눕니다.
3. 코덱스에는 `STYLE_GUIDE.md`의 마지막 “코덱스 프롬프트용 요약”을 우선 넣고 시작하면 편합니다.
4. 구현 후 이미지와 비교하며 간격, 라운드, 카드 구조를 맞추면 됩니다.

## 추천 구현 범위
- 홈 화면
- 전체 메뉴 화면
- 계산기 상세
- 길이 변환 상세
- 공통 헤더/사이드바/카드 컴포넌트

## 비고
이 자료는 시각 스타일과 화면 구조 기준 문서예요.
실제 계산/변환 로직은 별도로 구현하면 됩니다.
